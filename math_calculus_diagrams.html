The Derivative as a Limit

Watching the secant slope converge to the tangent slope as h → 0, for f(x) = x² at x = 2

Step 1 of 5

The parabola f(x) = x². We want the instantaneous rate of change — the derivative — at point P = (2, 4).

Secant through P = (2, 4) and Q = (4, 16) with h = 2. Slope = (16 − 4) / 2 = 6. A rough first estimate of f′(2).

Move Q to (3, 9), h = 1. Slope = (9 − 4) / 1 = 5. The secant rotates as Q approaches P — getting closer.

Q = (2.5, 6.25), h = 0.5. Slope = 4.5. Q is almost on top of P — the secant and tangent are nearly the same line.

As h → 0 the secant converges to the tangent line, slope 4. This is the derivative: f′(2) = 2·2 = 4 — the exact instantaneous rate of change at P.

Linear Approximation — Estimating √4.1

Chapter 7's first worked example, step by step: use the tangent line at a = 4 to estimate √4.1 without a calculator

Step 1 of 4

We want to estimate √4.1 by hand. Start with the curve f(x) = √x and the nearby point where the exact value is easy: a = 4, f(4) = 2.

The tangent line at (4, 2) has slope f′(4) = 1/(2√4) = 1/4. That gives the linear approximation L(x) = 2 + (1/4)(x − 4) — the best straight-line stand-in for f near x = 4.

Evaluate the tangent line at x = 4.1: L(4.1) = 2 + (1/4)(0.1) = 2.025. That single multiplication is the entire estimate — no square root needed.

The true value √4.1 ≈ 2.0248 sits just a hair below the tangent line's estimate — since √x is concave down, the tangent slightly overestimates. The linear approximation is accurate to within 0.00015, using nothing but a slope and a line.
