Chapter 1

Variables & Expressions

A variable is a letter that stands for an unknown number. An algebraic expression combines variables and numbers using arithmetic operations. To evaluate an expression, substitute a number for each variable and simplify.

Like terms share the same variable raised to the same power. Combine like terms by adding or subtracting their coefficients: $5x + 3x - 2x = (5 + 3 - 2)x = 6x$.

The distributive property lets you multiply a factor through a sum: $a(b + c) = ab + ac$.

Key rules
$$a(b + c) = ab + ac \qquad \text{(distributive)}$$ $$ax + bx = (a+b)x \qquad \text{(combine like terms)}$$
Worked example — evaluate $f(x) = 2x + 3$ at $x = 4$
$f(4) = 2(4) + 3$substitute $x = 4$
$= 8 + 3$multiply first
$= 11$add
Worked example — expand $3(x + 4)$
$3 \cdot x + 3 \cdot 4$distribute the 3
$= 3x + 12$simplify
Chapter 2

Linear Equations

A linear equation in one variable has the form $ax + b = c$ with $a \ne 0$. The goal is to isolate the variable by performing the same operation on both sides.

One-step equations need a single operation: add, subtract, multiply, or divide both sides. Multi-step equations require clearing constants first, then dividing by the coefficient. When the variable appears on both sides, move all variable terms to one side by adding or subtracting.

Solving strategy

1. Distribute and combine like terms on each side.
2. Move variable terms to one side; constants to the other.
3. Divide both sides by the coefficient of the variable.

Worked example — solve $3x + 5 = 20$
$3x = 15$subtract 5 from both sides
$x = 5$divide both sides by 3
Worked example — solve $5x - 2 = 3x + 10$
$2x - 2 = 10$subtract $3x$ from both sides
$2x = 12$add 2 to both sides
$x = 6$divide by 2
Worked example — solve $\dfrac{x}{2} + \dfrac{x}{3} = 5$
$\dfrac{3x}{6} + \dfrac{2x}{6} = 5$LCD = 6
$\dfrac{5x}{6} = 5$add fractions
$x = 6$multiply both sides by $\frac{6}{5}$
Chapter 3

Lines & Slope

The slope of a line through points $(x_1, y_1)$ and $(x_2, y_2)$ measures its steepness: rise over run. Slope-intercept form $y = mx + b$ directly gives the slope $m$ and the $y$-intercept $b$ (where the line crosses the $y$-axis).

Key formulas
$$m = \frac{y_2 - y_1}{x_2 - x_1} \qquad \text{slope}$$ $$y = mx + b \qquad \text{slope-intercept form}$$
Worked example — slope through $(1, 2)$ and $(3, 8)$
$m = \dfrac{8 - 2}{3 - 1}$slope formula
$= \dfrac{6}{2} = 3$simplify
Worked example — write the equation with $m = 3$, $b = 2$
$y = 3x + 2$substitute $m$ and $b$ into $y = mx + b$
Chapter 4

Systems of Equations

A system of equations is two or more equations with the same variables. The solution is the point $(x, y)$ that satisfies all equations simultaneously. Two main algebraic methods:

Elimination: Add (or subtract) the equations to cancel one variable. Multiply one or both equations by a constant first if needed. Substitution: Solve one equation for one variable, then substitute that expression into the other equation.

Worked example (elimination) — $x + y = 8$ and $x - y = 2$
$2x = 10$add both equations; $y$ cancels
$x = 5$divide by 2
$5 + y = 8 \Rightarrow y = 3$back-substitute
Worked example (substitution) — $x + 2y = 7$ and $3x - y = 7$
$y = 3x - 7$solve second eq. for $y$
$x + 2(3x-7) = 7$substitute into first eq.
$7x = 21 \Rightarrow x = 3$simplify and solve
$y = 3(3) - 7 = 2$back-substitute
Chapter 5

Factoring

Factoring rewrites a polynomial as a product of simpler polynomials. It is the reverse of expanding.

GCF: Factor out the greatest common factor first. Difference of squares: $a^2 - b^2 = (a+b)(a-b)$. Trinomials $x^2 + bx + c$: find two numbers that multiply to $c$ and add to $b$. Leading coefficient ≠ 1 ($ax^2 + bx + c$): use the AC method — find two numbers multiplying to $ac$ that add to $b$, then factor by grouping.

Key patterns
$$a^2 - b^2 = (a+b)(a-b)$$ $$x^2 + bx + c = (x + p)(x + q) \quad \text{where } p+q=b,\ pq=c$$
Worked example — factor $x^2 + 2x - 8$
find $p, q$: $pq = -8$, $p+q = 2$$p=4$, $q=-2$
$(x + 4)(x - 2)$write the factored form
Worked example — factor $2x^2 + 5x - 3$
$ac = 2 \cdot (-3) = -6$; find $p, q$: $pq=-6$, $p+q=5$$p=6$, $q=-1$
$2x^2 + 6x - x - 3$rewrite middle term
$2x(x + 3) - 1(x + 3)$factor by grouping
$(2x - 1)(x + 3)$factor out $(x+3)$
Chapter 6

Quadratic Equations

A quadratic equation has the form $ax^2 + bx + c = 0$. Three main solution methods:

Factoring: Factor the left side; use the zero-product property ($AB = 0 \Rightarrow A = 0$ or $B = 0$). Completing the square: Rewrite as $(x + h)^2 = k$, then take square roots. Quadratic formula: Always works — use when factoring is not obvious.

Quadratic formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Vertex form (completing the square)
$$y = a(x - h)^2 + k \quad \text{vertex at } (h, k)$$
Worked example (factoring) — solve $x^2 - 5x + 6 = 0$
$(x - 2)(x - 3) = 0$factors: $-2 \cdot -3 = 6$, $-2 + -3 = -5$
$x = 2$ or $x = 3$zero-product property
Worked example (completing the square) — solve $x^2 + 6x - 7 = 0$
$x^2 + 6x = 7$move constant
$x^2 + 6x + 9 = 16$add $(6/2)^2 = 9$ to both sides
$(x + 3)^2 = 16$perfect square
$x + 3 = \pm 4$take square root
$x = 1$ or $x = -7$solve each case
Worked example (FOIL expansion) — expand $(2x - 3)(x + 4)$
$2x \cdot x + 2x \cdot 4 + (-3) \cdot x + (-3) \cdot 4$FOIL
$2x^2 + 8x - 3x - 12$multiply each pair
$2x^2 + 5x - 12$combine like terms
Chapter 7

Rational Expressions

A rational expression is a fraction whose numerator and denominator are polynomials. Simplify by factoring both and cancelling common factors. The cancelled factors create restrictions on the domain (values that make the denominator zero are excluded).

To solve an equation with rational expressions, multiply both sides by the LCD (least common denominator) to clear all fractions.

Worked example — simplify $\dfrac{x^2 - 9}{x + 3}$
$\dfrac{(x+3)(x-3)}{x+3}$factor numerator: difference of squares
$x - 3$cancel $(x+3)$, $x \ne -3$
Worked example — solve $\dfrac{x+2}{x-2} = 3$
$x + 2 = 3(x - 2)$multiply both sides by $(x-2)$
$x + 2 = 3x - 6$distribute
$8 = 2x \Rightarrow x = 4$solve the linear equation
Chapter 8

Radicals

The square root $\sqrt{a}$ is the non-negative number whose square equals $a$. To solve an equation containing a square root, isolate the radical and then square both sides. Always check your solution — squaring can introduce extraneous roots.

Key rule
$$\text{If } \sqrt{x} = k \text{ (and } k \ge 0\text{), then } x = k^2$$
Worked example — solve $\sqrt{x} = 8$
$(\sqrt{x})^2 = 8^2$square both sides
$x = 64$check: $\sqrt{64} = 8$ ✓
Worked example — solve $x^3 = 8$
$x = \sqrt[3]{8}$take the cube root of both sides
$x = 2$since $2^3 = 8$
Chapter 9

Absolute Value Equations

The absolute value $|a|$ gives the distance from $a$ to zero, so it is always non-negative. The equation $|A| = b$ (with $b \ge 0$) has two cases: the expression inside equals $b$ or $-b$.

Key rule
$$|A| = b \implies A = b \quad \text{or} \quad A = -b \qquad (b \ge 0)$$
Worked example — solve $|2x - 3| = 7$
$2x - 3 = 7$case 1
$x = 5$add 3, divide by 2
$2x - 3 = -7$case 2
$x = -2$add 3, divide by 2
Chapter 10

Exponentials & Logarithms

An exponential equation has the variable in the exponent. If both sides can be written with the same base, set the exponents equal. A logarithm is the inverse of exponentiation: $\log_b(x) = y$ means $b^y = x$.

Key identities
$$b^x = b^y \implies x = y \qquad \text{(same base)}$$ $$\log_b(x) = y \iff b^y = x \qquad \text{(definition)}$$ $$\log_b(b^x) = x \quad \text{and} \quad b^{\log_b x} = x$$
Worked example — solve $3^x = 27$
$3^x = 3^3$write $27 = 3^3$
$x = 3$exponents must be equal
Worked example — solve $\log_3(x + 1) = 2$
$3^2 = x + 1$rewrite in exponential form
$9 = x + 1$evaluate $3^2$
$x = 8$subtract 1
Worked example — solve $e^{2x} = e^6$
$2x = 6$same base $e$; set exponents equal
$x = 3$divide by 2
Chapter 11

Functions & Inverses

A function $f$ assigns exactly one output to each input. The notation $f(x)$ means "the value of $f$ at $x$." To evaluate, replace every $x$ with the given input and simplify.

The inverse function $f^{-1}$ reverses the mapping of $f$: if $f(a) = b$ then $f^{-1}(b) = a$. To find $f^{-1}$, swap $x$ and $y$ in $y = f(x)$, then solve for $y$.

Finding an inverse

1. Replace $f(x)$ with $y$.
2. Swap every $x$ and $y$.
3. Solve for $y$.
4. Write the result as $f^{-1}(x)$.

Worked example — find $f^{-1}$ for $f(x) = 3x - 6$
$y = 3x - 6$write with $y$
$x = 3y - 6$swap $x$ and $y$
$x + 6 = 3y$add 6 to both sides
$f^{-1}(x) = \dfrac{x+6}{3}$divide by 3
