Chapter 1

Real Numbers & Properties

Numbers are organized into sets. Natural numbers are the counting numbers $1, 2, 3, \ldots$ Whole numbers add zero: $0, 1, 2, 3, \ldots$ Integers extend in both directions: $\ldots, -2, -1, 0, 1, 2, \ldots$ Rational numbers are ratios of integers ($\frac{p}{q}$, $q \ne 0$); they include all terminating and repeating decimals. Irrational numbers cannot be written as fractions — e.g., $\sqrt{2}$ and $\pi$. Together they form the real numbers.

The absolute value $|a|$ is the distance from $a$ to zero on the number line: $|5| = 5$ and $|-5| = 5$.

Order of operations (PEMDAS): Parentheses → Exponents → Multiplication and Division (left to right) → Addition and Subtraction (left to right).

Properties of real numbers (for all real $a, b, c$)
$$a + b = b + a \quad \text{(commutative: addition)}$$ $$ab = ba \quad \text{(commutative: multiplication)}$$ $$(a + b) + c = a + (b + c) \quad \text{(associative: addition)}$$ $$a(bc) = (ab)c \quad \text{(associative: multiplication)}$$ $$a(b + c) = ab + ac \quad \text{(distributive)}$$ $$a + 0 = a, \quad a \cdot 1 = a \quad \text{(identity)}$$ $$a + (-a) = 0, \quad a \cdot \tfrac{1}{a} = 1 \ (a \ne 0) \quad \text{(inverse)}$$
Worked example — evaluate $2 + 3 \times 4$
$2 + 3 \times 4$multiply before adding (PEMDAS)
$= 2 + 12$$3 \times 4 = 12$
$= 14$add
Worked example — evaluate $3^2 - (4 + 1) \cdot 2$
$3^2 - 5 \cdot 2$parentheses first: $4+1=5$
$= 9 - 10$exponent $3^2=9$; multiply $5 \cdot 2=10$
$= -1$subtract
Chapter 2

Variables & Expressions

A variable is a letter that stands for an unknown number. An algebraic expression combines variables and numbers using arithmetic. To evaluate an expression, substitute a value for each variable and simplify using the order of operations.

Like terms share the same variable raised to the same power. Combine them by adding or subtracting their coefficients: $5x + 3x - 2x = (5+3-2)x = 6x$. The distributive property lets you multiply through a sum or difference: $a(b + c) = ab + ac$.

Key rules
$$a(b + c) = ab + ac \qquad \text{(distributive)}$$ $$ax + bx = (a+b)x \qquad \text{(combine like terms)}$$
Worked example — evaluate $f(x) = 2x + 3$ at $x = 4$
$f(4) = 2(4) + 3$substitute $x = 4$
$= 8 + 3 = 11$multiply, then add
Worked example — expand $3(x + 4)$
$3 \cdot x + 3 \cdot 4 = 3x + 12$distribute 3 to each term
Worked example — simplify $5x + 3x - 2x$
$(5 + 3 - 2)x = 6x$all terms are like terms; combine coefficients
Chapter 3

Linear Equations

A linear equation in one variable has the form $ax + b = c$ with $a \ne 0$. Solve by isolating the variable — perform the same operation on both sides to keep the equation balanced.

Solving strategy

1. Distribute and combine like terms on each side.
2. Move all variable terms to one side; all constants to the other.
3. Divide both sides by the coefficient of the variable.

Worked example — solve $3x + 5 = 20$
$3x = 15$subtract 5 from both sides
$x = 5$divide both sides by 3
Worked example — solve $5x - 2 = 3x + 10$ (variable on both sides)
$2x - 2 = 10$subtract $3x$ from both sides
$2x = 12 \Rightarrow x = 6$add 2; divide by 2
Worked example — solve $\dfrac{x}{2} + \dfrac{x}{3} = 5$ (fractional equation)
$\dfrac{3x + 2x}{6} = 5$LCD = 6; rewrite fractions
$5x = 30 \Rightarrow x = 6$multiply both sides by 6; divide by 5
Chapter 4

Linear Inequalities

Solve a linear inequality the same way as a linear equation, with one critical exception: reverse the inequality sign when you multiply or divide both sides by a negative number.

Solutions are often written in interval notation: parentheses for strict inequalities ($<$, $>$) and brackets for non-strict ($\leq$, $\geq$). For example, $x > 3$ is $(3, \infty)$ and $x \leq 5$ is $(-\infty, 5]$.

A compound inequality combines two inequalities with "and" ($a < x < b$, meaning $x$ is between $a$ and $b$) or "or" ($x < a$ or $x > b$). Solve "and" compound inequalities by operating on all three parts simultaneously.

Key rule
$$\text{If } a < b \text{ and } c < 0, \text{ then } ac > bc \quad \text{(flip the sign!)}$$
Worked example — solve $x + 4 > 9$
$x > 9 - 4$subtract 4 from both sides
$x > 5$, i.e., $(5, \infty)$simplify
Worked example — solve $3x - 2 \leq 10$
$3x \leq 12$add 2 to both sides
$x \leq 4$, i.e., $(-\infty, 4]$divide by 3 (positive — no flip)
Worked example — solve $-2x + 1 > 7$
$-2x > 6$subtract 1
$x < -3$divide by $-2$ — reverse the sign
Worked example — solve compound $-1 \leq 2x + 3 \leq 7$
$-4 \leq 2x \leq 4$subtract 3 from all three parts
$-2 \leq x \leq 2$, i.e., $[-2, 2]$divide all three parts by 2
Chapter 5

Graphing Linear Equations

Every point in the plane is given by an ordered pair $(x, y)$. The x-intercept is where the line crosses the x-axis (set $y = 0$); the y-intercept is where it crosses the y-axis (set $x = 0$).

Slope measures steepness: rise over run. Slope-intercept form $y = mx + b$ directly shows slope $m$ and y-intercept $b$. Standard form $Ax + By = C$ can be converted to slope-intercept by solving for $y$.

Two lines are parallel if they have equal slopes ($m_1 = m_2$) and perpendicular if their slopes are negative reciprocals ($m_1 \cdot m_2 = -1$).

Key formulas
$$m = \frac{y_2 - y_1}{x_2 - x_1} \qquad \text{slope between two points}$$ $$y = mx + b \qquad \text{slope-intercept form}$$ $$Ax + By = C \qquad \text{standard form}$$
Worked example — slope through $(1, 2)$ and $(3, 8)$
$m = \dfrac{8-2}{3-1} = \dfrac{6}{2} = 3$slope formula
Worked example — y-intercept of $y = 3x + 5$
set $x = 0$: $y = 3(0) + 5 = 5$y-intercept is $(0, 5)$
Worked example — x-intercept of $3x - 2y = 12$
set $y = 0$: $3x = 12 \Rightarrow x = 4$x-intercept is $(4, 0)$
Worked example — convert $2x + 4y = 12$ to slope-intercept form
$4y = -2x + 12$subtract $2x$ from both sides
$y = -\dfrac{x}{2} + 3$divide both sides by 4; slope $= -\frac{1}{2}$, y-int $= 3$
Chapter 6

Systems of Equations

A system of linear equations is two or more equations sharing the same variables. The solution is the ordered pair $(x, y)$ satisfying all equations. Systems can have one solution (lines intersect), no solution (parallel lines), or infinitely many solutions (same line).

Elimination: Multiply equations by constants so that adding them cancels one variable. Substitution: Solve one equation for one variable and substitute into the other.

Worked example (elimination) — $x + y = 8$ and $x - y = 2$
$2x = 10 \Rightarrow x = 5$add both equations; $y$ cancels
$5 + y = 8 \Rightarrow y = 3$back-substitute
Worked example (substitution) — $x + 2y = 7$ and $3x - y = 7$
$y = 3x - 7$solve second equation for $y$
$x + 2(3x-7) = 7 \Rightarrow 7x = 21 \Rightarrow x = 3$substitute into first; solve
$y = 3(3) - 7 = 2$back-substitute

When to use each method: Use substitution when one equation is already solved for a variable, or when isolating a variable introduces no messy fractions. Use elimination when both equations are in standard form and the coefficients of one variable match (or are easy to match by multiplying). Use graphing for visual intuition only — it is unreliable when the intersection falls at non-integer coordinates.

Worked example (word problem) — adult tickets $12, child tickets $7; 250 sold for $2,400
Let $a$ = adult tickets, $c$ = child ticketsdefine variables
$a + c = 250$ and $12a + 7c = 2400$total count and total revenue
$a = 250 - c$solve first equation for $a$
$12(250 - c) + 7c = 2400$substitute into revenue equation
$3000 - 12c + 7c = 2400 \Rightarrow -5c = -600$distribute; simplify
$c = 120$, $\; a = 250 - 120 = 130$divide by $-5$; back-substitute
Chapter 7

Polynomials

A polynomial is a sum of terms of the form $ax^n$ where $n$ is a non-negative integer. The degree is the highest exponent. Special cases: monomial (1 term), binomial (2 terms), trinomial (3 terms).

Add/subtract polynomials by combining like terms. Multiply using the distributive property; for two binomials use FOIL (First, Outer, Inner, Last).

Special products appear so often they are worth memorizing:

Special product patterns
$$(a + b)^2 = a^2 + 2ab + b^2 \qquad \text{(perfect square trinomial)}$$ $$(a - b)^2 = a^2 - 2ab + b^2 \qquad \text{(perfect square trinomial)}$$ $$(a + b)(a - b) = a^2 - b^2 \qquad \text{(difference of squares)}$$
Worked example — add $(2x + 3) + (4x - 1)$
$2x + 4x + 3 - 1$remove parentheses
$= 6x + 2$combine like terms
Worked example — subtract $(4x^3 - 2x^2 + x) - (x^3 + 3x^2 - 4x)$
$4x^3 - 2x^2 + x - x^3 - 3x^2 + 4x$distribute the minus sign
$= 3x^3 - 5x^2 + 5x$combine like terms
Worked example — expand $(x + 3)^2$
$x^2 + 2 \cdot x \cdot 3 + 3^2$$(a+b)^2 = a^2 + 2ab + b^2$, $a=x$, $b=3$
$= x^2 + 6x + 9$simplify
Worked example — expand $(3x - 2)(3x + 2)$
$(3x)^2 - 2^2$$(a-b)(a+b) = a^2 - b^2$
$= 9x^2 - 4$compute each square
Worked example — FOIL $(2x - 3)(x + 4)$
$2x^2 + 8x - 3x - 12$First $2x \cdot x$, Outer $2x \cdot 4$, Inner $-3 \cdot x$, Last $-3 \cdot 4$
$= 2x^2 + 5x - 12$combine like terms
Chapter 8

Factoring

Factoring rewrites a polynomial as a product of simpler factors — the reverse of expanding. Always check for a greatest common factor (GCF) first.

For trinomials $x^2 + bx + c$: find two numbers $p, q$ with $pq = c$ and $p + q = b$. For $ax^2 + bx + c$ with $a \ne 1$: use the AC method — find $p, q$ with $pq = ac$ and $p + q = b$, then split the middle term and factor by grouping.

Key patterns
$$a^2 - b^2 = (a+b)(a-b) \qquad \text{(difference of squares)}$$ $$a^3 - b^3 = (a-b)(a^2+ab+b^2) \qquad \text{(difference of cubes)}$$ $$a^3 + b^3 = (a+b)(a^2-ab+b^2) \qquad \text{(sum of cubes)}$$ $$x^2 + bx + c = (x+p)(x+q), \quad pq=c,\ p+q=b$$
Worked example — factor $x^2 + 2x - 8$
find $p, q$: $pq = -8$, $p+q = 2$$p=4$, $q=-2$ works
$(x + 4)(x - 2)$write the factored form
Worked example — factor $2x^2 + 5x - 3$ (AC method)
$ac = -6$; find $p, q$: $pq=-6$, $p+q=5$$p=6$, $q=-1$
$2x^2 + 6x - x - 3$rewrite middle term
$2x(x+3) - 1(x+3)$factor by grouping
$(2x-1)(x+3)$factor out $(x+3)$
Chapter 9

Quadratic Equations

A quadratic equation has the form $ax^2 + bx + c = 0$ with $a \ne 0$. Four solution methods:

1. Factoring + zero-product property: $AB = 0 \Rightarrow A = 0$ or $B = 0$. 2. Square root property: If $x^2 = k$ then $x = \pm\sqrt{k}$. 3. Completing the square: Rewrite as $(x + h)^2 = k$. 4. Quadratic formula: Always works regardless of the discriminant.

The discriminant $b^2 - 4ac$ is a quick check before you solve. If $b^2 - 4ac > 0$: two distinct real roots (parabola crosses the x-axis twice). If $b^2 - 4ac = 0$: one repeated real root (parabola just touches the x-axis at its vertex). If $b^2 - 4ac < 0$: two complex conjugate roots — no real solutions (parabola does not cross the x-axis). Evaluate the discriminant first to know what kind of answer to expect.

Quadratic formula
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Vertex form
$$y = a(x - h)^2 + k \quad \text{vertex at } (h, k)$$

Vertex form $f(x) = a(x-h)^2 + k$ reveals the vertex $(h, k)$ directly — no solving required. If $a < 0$ the parabola opens downward and the vertex is a maximum; if $a > 0$ it is a minimum. This makes vertex form the natural choice for maximum/minimum problems (peak height, maximum profit). Convert from standard form by completing the square, or use the shortcut $h = -\tfrac{b}{2a}$ and $k = f(h)$.

Worked example (factoring) — solve $x^2 - 5x + 6 = 0$
$(x-2)(x-3) = 0$$pq=6$, $p+q=-5$: $p=-2$, $q=-3$
$x = 2$ or $x = 3$zero-product property
Worked example (quadratic formula) — solve $x^2 - 6x + 8 = 0$
$x = \dfrac{6 \pm \sqrt{36 - 32}}{2}$$a=1, b=-6, c=8$
$= \dfrac{6 \pm \sqrt{4}}{2} = \dfrac{6 \pm 2}{2}$discriminant $= 4$, $\sqrt{4}=2$
$x = 4$ or $x = 2$evaluate both cases
Worked example (completing the square) — solve $x^2 + 6x - 7 = 0$
$x^2 + 6x = 7$move constant
$x^2 + 6x + 9 = 16$add $(6/2)^2 = 9$ to both sides
$(x+3)^2 = 16 \Rightarrow x+3 = \pm 4$perfect square; square root property
$x = 1$ or $x = -7$solve each case
Worked example (vertex by completing the square) — $y = x^2 - 6x + 5$
$y = (x^2 - 6x + 9) + 5 - 9$add and subtract 9
$y = (x-3)^2 - 4$vertex form; vertex at $(3, -4)$
Worked example (maximum height) — $h(t) = -16t^2 + 64t + 5$
$t = -\dfrac{b}{2a} = -\dfrac{64}{2(-16)} = \dfrac{-64}{-32} = 2$ secondstime of maximum: $h = -\tfrac{b}{2a}$; $a=-16$, $b=64$
$h(2) = -16(4) + 64(2) + 5$substitute $t = 2$
$= -64 + 128 + 5 = 69$ feetmaximum height occurs at the vertex
Chapter 10

Rational Expressions & Equations

A rational expression is a fraction with polynomial numerator and denominator. Simplify by factoring both and cancelling common factors; note the values of the variable that make the denominator zero are excluded from the domain.

To solve a rational equation, multiply both sides by the LCD to clear all denominators, then solve the resulting polynomial equation. Always check for extraneous solutions.

Worked example — simplify $\dfrac{x^2 - 9}{x + 3}$
$\dfrac{(x+3)(x-3)}{x+3}$factor numerator
$= x - 3 \quad (x \ne -3)$cancel $(x+3)$
Worked example — simplify $\dfrac{x^2+3x+2}{x^2+x-2}$
$\dfrac{(x+1)(x+2)}{(x+2)(x-1)}$factor numerator and denominator
$= \dfrac{x+1}{x-1} \quad (x \ne -2)$cancel $(x+2)$
Worked example — solve $\dfrac{x+2}{x-2} = 3$
$x + 2 = 3(x-2)$multiply both sides by $(x-2)$
$x + 2 = 3x - 6$distribute
$8 = 2x \Rightarrow x = 4$solve; check: $\frac{6}{2}=3$ ✓

Radicals & Rational Exponents

The principal square root $\sqrt{a}$ is the non-negative number whose square is $a$. The $n$th root $\sqrt[n]{a}$ satisfies $(\sqrt[n]{a})^n = a$.

Rational exponents connect roots and powers: $a^{1/n} = \sqrt[n]{a}$ and $a^{m/n} = (\sqrt[n]{a})^m = \sqrt[n]{a^m}$. To solve an equation with a radical, isolate it and raise both sides to the appropriate power. Always check for extraneous solutions when squaring.

Key identities
$$a^{1/n} = \sqrt[n]{a} \qquad a^{m/n} = \left(\sqrt[n]{a}\right)^m$$ $$\sqrt{ab} = \sqrt{a}\,\sqrt{b} \qquad \sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$$
Worked example — solve $\sqrt{x} = 8$
$x = 8^2 = 64$square both sides; check $\sqrt{64}=8$ ✓
Worked example — solve $x^3 = 8$
$x = \sqrt[3]{8} = 2$take the cube root; $2^3 = 8$ ✓
Worked example — solve $x^{2/3} = 4$
$x = 4^{3/2} = (\sqrt{4})^3$raise both sides to the $3/2$ power
$= 2^3 = 8$$\sqrt{4} = 2$, then cube it

Absolute Value

$|a|$ gives the distance from $a$ to zero. An absolute value equation $|A| = b$ ($b \geq 0$) splits into two cases: $A = b$ or $A = -b$.

An absolute value inequality also splits into two cases, but the direction depends on the inequality symbol:

Key rules
$$|A| = b \implies A = b \text{ or } A = -b \qquad (b \geq 0)$$ $$|A| < b \implies -b < A < b \qquad \text{("and")}$$ $$|A| > b \implies A < -b \text{ or } A > b \qquad \text{("or")}$$
Worked example (equation) — solve $|2x - 3| = 7$
$2x - 3 = 7 \Rightarrow x = 5$case 1
$2x - 3 = -7 \Rightarrow x = -2$case 2
Worked example (inequality) — solve $|2x - 1| \leq 5$
$-5 \leq 2x - 1 \leq 5$$|A| \leq b$ means $-b \leq A \leq b$
$-4 \leq 2x \leq 6$add 1 to all parts
$-2 \leq x \leq 3$divide by 2; solution is $[-2, 3]$

Exponentials & Logarithms

An exponential equation has the variable in the exponent. If both sides share the same base, set the exponents equal. A logarithm is the inverse of exponentiation: $\log_b x = y$ means $b^y = x$. The natural logarithm $\ln x = \log_e x$ uses base $e \approx 2.718$.

Key identities
$$b^x = b^y \implies x = y \qquad \text{(same-base rule)}$$ $$\log_b(xy) = \log_b x + \log_b y \qquad \text{(product rule)}$$ $$\log_b\!\left(\tfrac{x}{y}\right) = \log_b x - \log_b y \qquad \text{(quotient rule)}$$ $$\log_b(x^r) = r\log_b x \qquad \text{(power rule)}$$ $$\log_b(b^x) = x \quad \text{and} \quad b^{\log_b x} = x$$
Worked example — solve $3^x = 27$
$3^x = 3^3$write $27 = 3^3$
$x = 3$same base → equal exponents
Worked example — solve $\log_3(x+1) = 2$
$3^2 = x + 1$rewrite in exponential form
$x = 8$$9 = x + 1$; subtract 1
Worked example — solve $e^{2x} = e^6$
$2x = 6 \Rightarrow x = 3$same base $e$; divide by 2

Exponential equations with mismatched bases: When both sides cannot be rewritten with the same base, take $\ln$ (or $\log$) of both sides. The power rule $\log_b(x^r) = r \log_b x$ brings the exponent down as a coefficient, converting the equation into a linear one you can solve directly.

Worked example — solve $5^{2x-1} = 100$
$(2x-1)\ln 5 = \ln 100$take $\ln$ of both sides; power rule brings down the exponent
$2x - 1 = \dfrac{\ln 100}{\ln 5} \approx 2.861$divide both sides by $\ln 5$
$2x \approx 3.861 \Rightarrow x \approx 1.93$add 1; divide by 2

Logarithmic equations: Use log laws to combine all logarithm terms into a single log on each side, then exponentiate both sides to eliminate the logarithm. Always check your solutions afterward — the argument of a logarithm must be strictly positive, so any candidate that produces a zero or negative argument is extraneous and must be rejected.

Worked example — solve $\log_2(x+3) + \log_2(x-1) = 5$
$\log_2[(x+3)(x-1)] = 5$product rule: sum of logs = log of product
$(x+3)(x-1) = 2^5 = 32$exponentiate both sides (base 2)
$x^2 + 2x - 3 = 32 \Rightarrow x^2 + 2x - 35 = 0$expand; move 32 to the left
$(x+7)(x-5) = 0 \Rightarrow x = -7$ or $x = 5$factor
$x = 5$ only$x=-7$: $\log_2(-4)$ undefined; $x=5$: $\log_2 8 + \log_2 4 = 3+2 = 5$ ✓

Functions & Inverses

A function $f$ assigns exactly one output to each input. The domain is the set of valid inputs; the range is the set of possible outputs. Function composition $(f \circ g)(x) = f(g(x))$ applies $g$ first, then feeds the result to $f$.

The inverse function $f^{-1}$ reverses $f$: if $f(a) = b$ then $f^{-1}(b) = a$. Find it by swapping $x$ and $y$ in $y = f(x)$ and solving for $y$. A function has an inverse only if it is one-to-one (passes the horizontal line test).

Finding an inverse

1. Replace $f(x)$ with $y$.  2. Swap $x$ and $y$.  3. Solve for $y$.  4. Write $f^{-1}(x)$.

Worked example (composition) — $(f \circ g)(2)$ where $f(x) = 3x-1$, $g(x) = x^2$
$g(2) = 2^2 = 4$evaluate $g$ first
$f(4) = 3(4) - 1 = 11$feed the result to $f$
Worked example (inverse) — find $f^{-1}$ for $f(x) = 3x - 6$
$x = 3y - 6$swap $x$ and $y$
$x + 6 = 3y$add 6 to both sides
$f^{-1}(x) = \dfrac{x+6}{3}$divide by 3

Domain restrictions: A function must be one-to-one (each output comes from exactly one input) to have an inverse on its full domain. Graphically, this means it passes the horizontal line test. If it fails — for example $f(x) = x^2$, where both $f(2)$ and $f(-2)$ equal 4 — you must restrict the domain first. Restricting $f(x) = x^2$ to $x \geq 0$ makes it one-to-one, and its inverse on that restricted domain is $f^{-1}(x) = \sqrt{x}$.

Verifying inverses with composition: $g$ is the inverse of $f$ if and only if both $f(g(x)) = x$ for every $x$ in the domain of $g$, and $g(f(x)) = x$ for every $x$ in the domain of $f$. Checking only one direction is not sufficient.

Worked example (verifying inverses) — $f(x) = 2x+3$ and $g(x) = \dfrac{x-3}{2}$
$f(g(x)) = 2 \cdot \dfrac{x-3}{2} + 3 = (x-3)+3 = x$ ✓substitute $g(x)$ into $f$
$g(f(x)) = \dfrac{(2x+3)-3}{2} = \dfrac{2x}{2} = x$ ✓substitute $f(x)$ into $g$
both compositions equal $x$ — confirmed inversesboth directions must hold
