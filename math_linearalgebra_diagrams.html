Dot Product — Component Method

How to calculate a · b for vectors a = (3, 2) and b = (1, 3), step by step.

Step 1 of 5

Two vectors a = (3, 2) (blue) and b = (1, 3) (green) drawn from the origin. The dot product a · b produces a single number.

Drop perpendiculars to the x-axis to read off the x-components: a₁ = 3 and b₁ = 1. Multiply: 3 × 1 = 3. This is the first partial product.

Draw horizontals to the y-axis: a₂ = 2 and b₂ = 3. Multiply: 2 × 3 = 6. This is the second partial product.

Add the two partial products: a · b = 3 + 6 = 9. The dot product of a = (3, 2) and b = (1, 3) is 9.

The dot product equals |a| |b| cos θ. Since a · b = 9 > 0, the angle θ ≈ 38° is less than 90° — the vectors point roughly the same way. The orange segment is the projection of b onto a.

Matrix × Vector — Row-by-Row Multiplication

How each entry of Av is built from one row of A and the whole vector v, for A = [[1,2],[3,4]] and v = (2, 1).

Step 1 of 4

A 2×2 matrix times a 2×1 vector produces a new 2×1 vector. Each output entry is the dot product of one matrix row with v.

Row 1 of A — (1, 2) — slides down to pair with v — (2, 1) — position by position. Multiply each pair, then add: 1×2 + 2×1 = 4. That becomes the first entry of the result.

Row 2 of A — (3, 4) — pairs with the same vector v. Multiply and add: 3×2 + 4×1 = 10. That becomes the second entry of the result.

The finished product: Av = (4, 10). Each entry is the dot product of one matrix row with v — exactly how a neural network layer computes y = Wx from one neuron's weight row and the input vector.

Gaussian Elimination — Solving a 2×2 System

Turn a system of equations into an augmented matrix, then use row operations to reduce it until the solution can be read straight off — for x + y = 5 and 2x + 3y = 13.

Step 1 of 4

Each equation becomes one row of an augmented matrix [A | b]: the coefficients of x and y, then a bar, then the constant. Solving the system means using row operations to simplify this matrix.

Use Row 1 (blue) as the pivot: multiply it by 2 — enough to match Row 2's x-coefficient — then subtract it from Row 2 (green), column by column. 2−2=0, 3−2=1, 13−10=3: Row 2 becomes [0, 1 | 3], which no longer has an x-term.

Now use Row 2 (blue) as the pivot: its multiplier is just 1 this time, since Row 1's y-coefficient is already 1. Subtract it from Row 1 (green), column by column. 1−0=1, 1−1=0, 5−3=2: Row 1 becomes [1, 0 | 2], which no longer has a y-term.

The matrix is now in reduced row echelon form — an identity matrix on the left — so the solution reads straight off the right column: x = 2 and y = 3. Both original equations check out.
