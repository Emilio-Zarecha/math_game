Three interactive diagrams: the handshake theorem, graph traversal (BFS vs DFS), and building a minimum spanning tree with Kruskal's algorithm.

1 — The Handshake Theorem

Click any faint line to toggle that edge on or off. Watch the degree of each vertex update live, and see that the sum of all degrees always equals 2|E| — no matter which edges you pick.

2 — Graph Traversal: BFS vs DFS

Pick a start vertex and a traversal method. Numbers show visit order; highlighted edges are the traversal tree — the edge used to first reach each vertex.

3 — Kruskal's Algorithm: Building an MST

Step through Kruskal's algorithm edge by edge, sorted by weight. Each edge is accepted (added to the tree) unless both endpoints are already connected, which would create a cycle.
